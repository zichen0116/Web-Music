# 🎵 World Execute Me - VS Code Style

一个基于 VS Code 界面风格的音乐可视化网页应用，灵感来自 [world-execute-me-idea](https://github.com/zmrlft/world-execute-me-idea)。

## ✨ 特性

### 🎨 VS Code 界面风格
- 完整的 VS Code 暗色主题界面
- 侧边栏、活动栏、状态栏、标签页系统
- 集成终端和面包屑导航

### 🐍 Python 代码主题
- Python 语法高亮
- Python 风格的终端输出

### ✨ 增强的视觉效果
- 打字机效果 + 光标闪烁
- 粒子动画背景
- 霓虹灯发光效果
- 音频波浪可视化
- Matrix 代码雨效果
- 心跳脉冲动画

### 🎮 增强的交互效果
- 键盘快捷键（空格暂停、Esc 停止）
- 可拖拽进度条
- 音量控制
- 全屏模式
- 亮色/暗色主题切换
- 鼠标悬停特效

### 📱 响应式设计
- 支持桌面和移动设备

## 🚀 使用方法

1. 将您的音频文件放入 `assets/audio/` 目录
2. 在 `js/main.js` 中配置音频路径
3. 打开 `index.html` 即可体验

## 📁 项目结构

```
/
├── index.html          # 主页面
├── css/
│   └── style.css       # 样式文件
├── js/
│   └── main.js         # JavaScript 逻辑
├── assets/
│   └── audio/          # 音频文件目录
└── README.md           # 项目说明
```

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Space` | 暂停/继续播放 |
| `Esc` | 停止播放 |
| `F11` | 全屏模式 |
| `M` | 静音/取消静音 |
| `←` / `→` | 快退/快进 5秒 |
